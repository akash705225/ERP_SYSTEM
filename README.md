# Employee Management System

A web-based Employee Management System built using Python, Flask, and MySQL. It allows administrators and HR personnel to add, view, update, and search for employee details through an interactive dashboard.

## Tech Stack
- **Backend:** Python 3.12, Flask 3.1.0
- **Database:** MySQL (using Flask-MySQLdb / mysqlclient)
- **Frontend:** HTML, CSS

## Prerequisites
- Python 3.x installed
- MySQL Server running locally 

## Installation & Setup

1. Make sure your MySQL server is running and your `project2` database is created along with the necessary tables (`admin`, `addemployee`, etc.).

2. Install the required Python dependencies:
   ```bash
   pip install -r required.txt
   ```

3. Run the Flask application:
   ```bash
   python app.py
   ```

4. Open your web browser and go to:
   `http://127.0.0.1:5000/`
